# 💪 BMI Calculator Website

## 📌 Project Description

This is a simple and interactive **BMI (Body Mass Index) Calculator** website built using HTML, CSS, and JavaScript.
It allows users to enter their height and weight and instantly calculates their BMI along with the health category.


## 🚀 Features

* User-friendly interface
* Instant BMI calculation
* Displays BMI category:

  * Underweight
  * Normal
  * Overweight
  * Obese
* Input validation (checks if fields are empty)
* Responsive and clean design


## 🛠️ Technologies Used

* HTML (Structure)
* CSS (Styling)
* JavaScript (Logic & Functionality)


## 📊 How It Works

1. User enters:

   * Height (in centimeters)
   * Weight (in kilograms)
2. The application:

   * Converts height into meters

   * Applies the BMI formula:

     BMI = weight / (height × height)

   * Displays the result along with the category


## 🧮 BMI Categories

| BMI Range   | Category    |
| ----------- | ----------- |
| < 18.5      | Underweight |
| 18.5 – 24.9 | Normal      |
| 25 – 29.9   | Overweight  |
| ≥ 30        | Obese       |

## ▶️ How to Run the Project

1. Download or clone the project
2. Open the project folder
3. Double-click on `index.html`
4. Enter height and weight
5. Click **Calculate**


## ⚠️ Notes

* Ensure both fields are filled before calculating
* Height must be entered in centimeters
* Weight must be entered in kilograms
